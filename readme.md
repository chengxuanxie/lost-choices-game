# 《迷途抉择》Lost Choices

> 第一人称视频互动游戏 - 在迷失中寻找真相

## 项目简介

《迷途抉择》是一款第一人称视频互动游戏，玩家将扮演一个在神秘设施中醒来的人，通过探索环境、与不同角色互动、做出关键选择来揭开自己的身世之谜。

游戏采用混合视觉方案：**关键帧图片 + 着色器效果** 为主力呈现方式，辅以 **短视频** 用于关键剧情节点，实现高质量视觉体验的同时有效控制制作成本。

### 技术栈

- **游戏引擎**: Godot 4.3+
- **开发语言**: GDScript
- **视觉方案**: 关键帧 + 着色器 + 短视频（混合方案）
- **存储服务**: 火山引擎 TOS

## 制作规范

### 一场景一参考图

每个场景使用 **1张核心参考图**，所有该场景的关键帧和短视频都基于这张参考图生成，确保视觉连贯性。

```
场景 = 1张核心参考图
    ↓
关键帧1, 关键帧2, ... → 基于参考图生成
短视频1, 短视频2, ... → 基于参考图生成
```

### 混合视觉方案

```
关键帧图片 + 着色器效果  ←  主力（成本节省87%）
        ↓
短视频  ←  仅关键剧情节点（~12个）
```
