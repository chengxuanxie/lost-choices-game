# 剧情节点查看器

交互式剧情结构可视化工具，方便查看各个节点的连接关系和详情。

## 使用方法

### 方式一：直接用浏览器打开
```bash
# 在浏览器中打开
tools/story_viewer/index.html
```

### 方式二：启动本地服务器
```bash
cd tools/story_viewer
python -m http.server 8080
# 然后访问 http://localhost:8080
```

## 功能说明

### 左侧面板 - 节点列表
- 搜索框：按节点ID或场景名搜索
- 过滤按钮：
  - 全部：显示所有节点
  - 选择：只显示有选择项的节点
  - 结局：只显示结局节点
  - 有视频：只显示有视频的节点
  - 断链：显示断开连接的节点

### 中间面板 - 连接图
- 树形结构显示节点关系
- 鼠标拖拽平移
- 滚轮缩放
- 点击节点选中
- 底部按钮：
  - 重置视图：回到初始位置
  - 切换标签：显示/隐藏节点文字

### 右侧面板 - 详情
显示选中节点的完整信息：
- 场景名、描述
- 视频路径和时长
- 关键帧数量
- 所有选择项及效果
- 下一节点链接
