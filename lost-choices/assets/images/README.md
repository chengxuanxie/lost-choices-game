# 图片资源目录

将游戏图片文件放置在此目录下。

## 目录结构

```
images/
├── backgrounds/   # 背景图片
├── characters/    # 角色立绘
├── ui/            # UI 元素
├── chapters/      # 章节缩略图
└── endings/       # 结局插图
```

## 支持的格式

- PNG (透明背景)
- WebP (压缩率高)
- JPG (背景图片)

## 图片规格

| 类型 | 分辨率 | 格式 |
|------|--------|------|
| 背景 | 1920x1080 | JPG/WebP |
| 角色立绘 | 根据设计 | PNG |
| UI 元素 | 根据设计 | PNG |
| 章节缩略图 | 320x180 | WebP/JPG |
| 结局插图 | 1920x1080 | WebP/JPG |
