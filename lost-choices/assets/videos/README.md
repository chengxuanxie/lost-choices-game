# 视频资源目录

将游戏视频文件放置在此目录下。

## 支持的格式

| 格式 | 编码 | 推荐浏览器 |
|------|------|------------|
| .ogv | Theora | Chrome, Firefox |
| .webm | VP9/VP8 | Safari, Chrome |
| .mp4 | H.264 | 所有浏览器（需要许可证）|

## 命名规范

视频文件命名格式: `ch1_sc001.ogv`

- `ch1` - 章节编号
- `sc001` - 场景编号

## 文件大小建议

- 单个视频建议不超过 50MB
- 总视频资源建议不超过 500MB（Web版本）

## 压缩设置

推荐视频编码设置：
- 分辨率: 1920x1080 或 1280x720
- 帧率: 24-30 fps
- 比特率: 2-5 Mbps
- 音频: AAC 128kbps
