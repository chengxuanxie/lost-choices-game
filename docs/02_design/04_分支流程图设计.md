## 一、分支系统概述

### 1.1 分支设计原则

| 原则 | 说明 |
|------|------|
| **有意义的选择** | 每个选择都应有实质性的影响 |
| **后果可见** | 玩家能感知到选择的后果 |
| **多线并行** | 不同好感度路线可同时发展 |
| **隐藏内容** | 高好感度解锁特殊选项和剧情 |
| **结局多样** | 不同路径导向不同结局 |

### 1.2 分支类型

| 类型 | 说明 | 示例 |
|------|------|------|
| **主线分支** | 影响剧情走向的重大选择 | 信任谁、去哪里 |
| **好感度分支** | 影响角色关系的对话选择 | 说话方式、帮助与否 |
| **条件分支** | 需要特定条件解锁的选择 | 好感度阈值、物品需求 |
| **隐藏分支** | 特殊条件触发的秘密剧情 | 特定选择组合 |

| 好感度范围 | 状态 | 解锁内容 |
|------------|------|----------|
| 0-20 | 警惕/疏离 | 基础互动选项 |
| 21-40 | 初步信任 | 额外对话选项、小剧情 |
| 41-60 | 信任/友好 | 个人故事、好感度标记 |
| 61-80 | 亲密/深信 | 特殊选项、专属剧情 |
| 81-100 | 深情/忠诚 | 最佳结局条件、隐藏内容 |
