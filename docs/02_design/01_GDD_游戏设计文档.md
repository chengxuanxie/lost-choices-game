## 📌 游戏概述

### 游戏名称
[待定] - 第一人称视频互动游戏

### 一句话描述
一款沉浸式第一人称视角的视频互动游戏，玩家通过选择对话选项影响剧情走向，体验独特的故事旅程。

### 游戏类型
- **主类型**: 互动电影 / 视频互动游戏
- **子类型**: 剧情冒险、选择驱动
- **视角**: 第一人称视角（POV）

### 目标受众
- 喜欢剧情向游戏的玩家
- 互动电影爱好者
- 寻求沉浸式体验的休闲玩家
- 年龄段：18-35岁

### 独特卖点 (USP)
1. **第一人称沉浸视角**: 玩家以主角视角体验故事，增强代入感
2. **真实视频拍摄**: 真人演员表演，电影级画面品质
3. **深度分支剧情**: 玩家选择真正影响故事走向和结局
4. **Web端即点即玩**: 无需下载，浏览器直接体验

## 🎯 核心玩法

### 玩法循环

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│   ┌─────────┐    ┌─────────┐    ┌─────────┐           │
│   │ 观看视频 │ ──▶│ 做出选择 │ ──▶│ 观看结果 │           │
│   └─────────┘    └─────────┘    └─────────┘           │
│        │              │              │                 │
│        │              │              │                 │
│        ▼              ▼              ▼                 │
│   ┌─────────┐    ┌─────────┐    ┌─────────┐           │
│   │ 了解剧情 │    │ 影响走向 │    │ 承担后果 │           │
│   └─────────┘    └─────────┘    └─────────┘           │
│                                                         │
│                      重复循环                            │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 核心机制

#### 1. 视频播放系统
- **无缝播放**: 视频片段之间平滑过渡
- **字幕支持**: 实时字幕显示
- **跳过功能**: 已观看内容可跳过
- **暂停/继续**: 随时控制播放进度

#### 2. 选择系统
- **时间限制选择**: 部分选择有时间限制，增加紧张感
- **隐藏选项**: 特定条件解锁隐藏选项
- **选择提示**: 通过视觉反馈提示玩家选择的影响
- **确认机制**: 重要选择前给予确认机会

#### 3. 状态系统
- **好感度系统**: 与NPC的关系影响剧情
- **道德值系统**: 玩家行为影响角色道德倾向
- **物品系统**: 收集的物品解锁新选择
- **记忆系统**: 角色记住玩家的选择和承诺

### 分支设计

#### 分支结构图
```
                    ┌── 选择A ──▶ 结局A
                    │
开始 ──▶ 第一章 ────┼── 选择B ──▶ 第二章B ──▶ 结局B
                    │
                    └── 选择C ──▶ 第二章C ──▶ 结局C
```

#### 分支设计原则
1. **选择有意义**: 每个选择都应有实际影响
2. **避免伪选择**: 不设计没有影响的"假"选择
3. **平衡分支**: 避免某条路线明显优于其他
4. **回溯激励**: 鼓励玩家尝试不同选择

## 🎮 游戏系统

### 对话系统

#### 对话选项设计
```
┌────────────────────────────────────────────────────────┐
│                                                        │
│   [NPC对话内容]                                        │
│   "你需要做出选择，这将决定一切..."                    │
│                                                        │
├────────────────────────────────────────────────────────┤
│                                                        │
│   ┌─────────────────────────────────────────────────┐ │
│   │  1. "我愿意帮助你"                    [好感度+] │ │
│   └─────────────────────────────────────────────────┘ │
│   ┌─────────────────────────────────────────────────┐ │
│   │  2. "我需要考虑一下"                   [中立]   │ │
│   └─────────────────────────────────────────────────┘ │
│   ┌─────────────────────────────────────────────────┐ │
│   │  3. "这不是我的事"                    [好感度-] │ │
│   └─────────────────────────────────────────────────┘ │
│   ┌─────────────────────────────────────────────────┐ │
│   │  4. [需要特定物品] ???                 [锁定]   │ │
│   └─────────────────────────────────────────────────┘ │
│                                                        │
│              ⏱ 剩余时间: 10秒                          │
│                                                        │
└────────────────────────────────────────────────────────┘
```

### 状态系统

#### 好感度系统
- **范围**: 0-100
- **等级**:
  - 敌对 (0-20): 角色对玩家充满敌意
  - 中立 (21-40): 角色对玩家保持距离
  - 友好 (41-70): 角色愿意帮助玩家
  - 亲密 (71-100): 角色完全信任玩家
- **影响**: 解锁不同对话选项、影响剧情走向

#### 道德值系统
- **范围**: -100 到 +100
- **倾向**:
  - 邪恶 (-100 到 -50)
  - 自私 (-49 到 -10)
  - 中立 (-9 到 9)
  - 善良 (10 到 49)
  - 英雄 (50 到 100)
- **影响**: 影响NPC态度、解锁不同结局

### 存档系统

#### 存档功能
- **手动存档**: 玩家可随时保存进度
- **自动存档**: 关键节点自动保存
- **多存档位**: 支持多个存档槽位
- **云存档**: 可选云端同步功能

#### 存档内容
- 当前剧情节点
- 所有状态值
- 获得的物品
- 已解锁的成就
- 选择历史记录

### 成就系统

| 成就 | 解锁条件 | 稀有度 |
|------|----------|--------|
| 初次选择 | 完成第一个选择 | 普通 |
| 探索者 | 解锁所有分支 | 稀有 |
| 完美结局 | 达成最佳结局 | 史诗 |
| 隐藏发现 | 发现隐藏内容 | 传说 |
