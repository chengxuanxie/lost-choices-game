## 一、世界观概述

### 故事背景

故事发生在现代都市，一个看似平静却暗流涌动的世界。

**核心设定**：
- 时间：现代（2026年）
- 地点：虚构的滨海城市"云海市"
- 核心冲突：真相与谎言的博弈，信任与背叛的选择

### 核心主题

> "每个选择都在定义你是谁。"

玩家将扮演主角，在迷失与觉醒之间做出抉择，揭开隐藏在表象之下的真相。

## 二、主要角色设定

### 2.1 主角（玩家视角）

| 项目 | 内容 |
|------|------|
| **角色定位** | 玩家扮演的角色，第一人称视角 |
| **代号** | "旅人"（非正式名称） |
| **年龄** | 约28岁（由玩家代入） |
| **身份** | 失忆者（故事开始时） |
| **性格** | 由玩家选择塑造 |

**背景设定**：
- 开场时在陌生环境中醒来，失去部分记忆
- 拥有某种被各方势力追逐的秘密
- 真实身份随着剧情推进逐渐揭晓

**角色弧光**：
```
迷失 → 怀疑 → 探索 → 觉醒 → 抉择
```

**玩家代入点**：
- 不设定具体姓名、性别、外貌，由玩家自我代入
- 通过第一人称视角增强沉浸感
- 性格和立场通过玩家选择体现
