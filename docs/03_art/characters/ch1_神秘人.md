## 二、SC-001中的角色表现

### 2.1 出场方式

| 要素 | 设计 |
|------|------|
| **媒介** | 房间扬声器/广播系统 |
| **先兆** | 沙沙声（5秒）→ 接通声（1秒）→ 人声 |
| **视觉** | 无（纯声音出场） |
| **距离感** | 声音从四面八方传来，无法定位 |

### 2.2 声音设计

```yaml
基础声音特征:
  性别: 无法辨认（经过处理）
  年龄: 无法判断
  语速: 中等偏慢，从容不迫
  音调: 略微压低，带有神秘感
  情感: 冷静、克制，偶尔透露紧迫感

特殊处理:
  - 变声效果: 明显的电子处理痕迹
  - 回响: 轻微的房间回响
  - 杂音: 偶尔的电子杂音
  - 关键词: 某些词语会被短暂干扰（如主角真名）

参考声线:
  - 类似《隐形守护者》中某些神秘角色的处理方式
  - 带有一定的机械感，但不是完全的机器人声音
```

### 2.3 台词风格

#### 说话特点
1. **信息控制**: 只透露最必要的信息
2. **称呼方式**: 始终称呼主角为"旅人"
3. **句式特点**: 短句为主，避免冗长解释
4. **隐喻使用**: 偶尔使用模糊的暗示

#### SC-001台词分析

| 台词 | 深层含义 | 暗示 |
|------|----------|------|
| "终于醒了，旅人。" | 一直在等待主角醒来 | 对主角非常了解 |
| "我只能告诉你最必要的事。" | 知道更多但选择不说 | 有更大的图谋 |
| "黄昏组织的安全屋" | 熟悉敌对组织的设施 | 可能是内鬼或前成员 |
| "72个小时" | 精确知道昏迷时间 | 一直在监控主角 |
| "每个选择都有代价" | 暗示游戏的核心主题 | 后续会有重要影响 |

## 五、角色形象设计

### 5.1 视觉设计原则

SC-001中神秘人**不出现视觉形象**，仅通过声音存在。这是刻意的设计选择：

1. **增强神秘感**: 未见其人先闻其声
2. **玩家代入**: 让玩家自己想象神秘人的样子
3. **悬念维持**: 为后续章节保留揭示空间
4. **恐怖谷效应**: 未知的恐惧比已知的更可怕

### 5.2 后续章节视觉建议

当神秘人最终揭示身份时：

| 章节 | 建议揭示程度 |
|------|--------------|
| 第二章 | 仍然只听到声音，但可能看到剪影 |
| 第三章 | 部分面部特征（如眼睛或侧脸） |
| 第四章 | 完整身份揭示 |

## 六、配音指导

### 6.1 声音处理流程

```
原始录音
    ↓
┌─────────────────────┐
│  基础变声处理        │
│  - 音调调整          │
│  - 速度微调          │
└─────────────────────┘
    ↓
┌─────────────────────┐
│  特效处理            │
│  - 房间回响          │
│  - 电话/广播滤波     │
│  - 轻微电子杂音      │
└─────────────────────┘
    ↓
┌─────────────────────┐
│  关键词处理          │
│  - 某些词语加干扰    │
│  - 增加神秘感        │
└─────────────────────┘
    ↓
最终效果
```

### 6.2 情感基调变化

| 时间点 | 情感基调 | 配合说明 |
|--------|----------|----------|
| 5:00-5:30 | 平静、观察 | 刚开始说话，建立存在感 |
| 5:30-6:10 | 严肃、紧迫 | 透露危险信息 |
| 6:20-7:00 | 中立、试探 | 给出选择，观察主角反应 |
| 7:00-7:30 | 意味深长 | 最后警告，留下悬念 |

### 6.3 配音参考示例

```
【正常语调】
"终于醒了，旅人。"
（停顿）
"你一定有很多问题。我是谁？这里是哪？我为什么会在这里？"
（轻微笑意）
"但我只能告诉你最必要的事。"

【严肃语调】
"听着，时间不多。"
（停顿，加重语气）
"你已经昏迷了72个小时。在你昏迷之前，你做了很多人不想让你做的事。"

【神秘语调】
"每个选择都有代价。"
（声音渐弱）
```
