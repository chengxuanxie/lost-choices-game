## 视频生成要点

### 第一人称视角要求

| 要素 | 具体要求 |
|------|----------|
| 起始位置 | 门口进入视角 |
| 运动轨迹 | 走向控制台 → 观察屏幕 → 操作通讯 |
| 运动速度 | 谨慎探索 |
| 手部可见 | 操作控制台时可见双手 |
| 画面稳定 | 稳定为主，减少晃动 |

### 分镜要点

1. **进入镜头**（5-10秒）
   - 推开门
   - 屏幕光线照亮环境

2. **观察镜头**（10-15秒）
   - 走向控制台
   - 环顾监控屏幕

3. **通讯镜头**（20-30秒）
   - 屏幕显示林晓薇
   - 操作通讯设备
