## 视频生成要点

### 第一人称视角要求

| 要素 | 具体要求 |
|------|----------|
| 起始位置 | 躺在床上视角 |
| 运动轨迹 | 缓慢坐起 → 环顾四周 → 起身 |
| 运动速度 | 缓慢、自然 |
| 手部可见 | 可见自己双手 |
| 画面稳定 | 轻微晃动模拟真实感 |

### 分镜要点

1. **开眼镜头**（5-10秒）
   - 模糊到清晰的视觉恢复
   - 天花板视角

2. **环顾镜头**（15-20秒）
   - 缓慢转头，扫视房间
   - 展示关键元素

3. **起身镜头**（10-15秒）
   - 坐起动作
   - 视角变化

### 情绪传递
- 通过光影营造神秘感
- 缓慢的运动增加紧张感
- 细节特写暗示线索
