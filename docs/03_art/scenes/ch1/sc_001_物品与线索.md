# SC-001 可收集物品与线索

> 神秘房间场景中的关键物品和线索设计

---

## 一、可收集物品

### 1.1 神秘纸条

| 属性 | 内容 |
|------|------|
| **物品ID** | paper_note |
| **获取方式** | 打开床头柜抽屉 |
| **获取时间** | 约2分钟（探索阶段） |
| **外观** | 泛黄的便签纸，边缘有些褶皱 |

#### 内容
```
749-2851

云海路32号，老地方见。
```

#### 主角反应
> "749...2851...这是密码吗？还是电话号码？"
> "云海路32号...老地方见？什么老地方？"

#### 用途
- **即时效果**: 触发主角对自身身份的好奇
- **SC-003**: 数字是保险箱密码，可获得追兵身份文件
- **SC-005**: 云海路32号是老张书店地址

---

### 1.2 钱包

| 属性 | 内容 |
|------|------|
| **物品ID** | wallet |
| **获取方式** | 打开床头柜抽屉 |
| **获取时间** | 约2分30秒（探索阶段） |
| **外观** | 黑色皮质钱包，有些磨损 |

#### 内容
- ❌ 身份证：缺失
- ✅ 现金：几张纸币（约200元）
- ✅ 照片：一张模糊的照片

#### 照片描述
一张被故意模糊处理的照片，隐约可见两个人影站在一起。照片边缘有烧焦的痕迹，似乎有人想销毁它但没有完全成功。

#### 主角反应
> "没有身份证..."
> "这张照片...模糊得看不清...照片里的人是我吗？"
> "为什么照片被烧过？发生了什么？"

#### 用途
- **即时效果**: 加深主角的身份困惑
- **第二章**: 照片真相关键道具，揭示主角与某角色的关系

---

### 1.3 记忆碎片（条件获取）

| 属性 | 内容 |
|------|------|
| **物品ID** | memory_fragment_01 |
| **获取方式** | 选择C "我想知道我是谁" |
| **获取时间** | 约7分钟（神秘人对话后） |
| **类型** | 隐藏收集品 |

#### 碎片内容
```
【记忆碎片 #1：代号】

...你听到一个声音在说...

"代号确认：旅人。记忆清除程序已启动。"
"目标：潜入黄昏组织核心。"
"时限：90天。"
"成功条件：获取'零号档案'。"

...然后是一片黑暗...
```

#### 主角反应
> "旅人...这是我的代号？"
> "记忆清除？我是被故意消除记忆的？"
> "零号档案...那是什么？"

#### 用途
- **收集系统**: 累计3个记忆碎片可触发隐藏剧情线
- **第二章**: 解锁额外对话选项
- **最终结局**: 影响真相揭示的完整度

## 二、环境线索

### 2.1 墙上符号

| 属性 | 内容 |
|------|------|
| **线索ID** | wall_symbol |
| **发现方式** | 仔细观察墙面 |
| **可见性** | 特殊墨水，特定角度可见 |
| **类型** | 伏笔线索 |

#### 符号描述
一个由三条弧线组成的图案，中心有一个圆点。符号用隐形墨水绘制，只有在暖光从特定角度照射时才能看到。

```
    ╲   ╱
     ╲ ╱
      ●
     ╱ ╲
    ╱   ╲
```

#### 主角反应
> "这是什么符号...用特殊墨水画的？"
> "为什么要在墙上画这个？是某种标记吗？"

#### 用途
- **即时效果**: 增加场景神秘感
- **SC-006**: 沈墨染的画作中出现相同符号
- **剧情揭示**: "黄昏"组织的标志

---

### 2.2 电子锁面板

| 属性 | 内容 |
|------|------|
| **线索ID** | electronic_lock |
| **发现方式** | 尝试开门 |
| **状态变化** | 红色 → 绿色 |
| **类型** | 剧情触发 |

#### 面板显示

**初始状态（探索阶段）**
```
┌─────────────────┐
│  ████████████   │
│  访问被拒绝     │
│  等待授权...    │
│  [红灯闪烁]     │
└─────────────────┘
```

**解锁状态（神秘人广播后）**
```
┌─────────────────┐
│  ████████████   │
│  授权已确认     │
│  欢迎您，旅人   │
│  [绿灯常亮]     │
└─────────────────┘
```

#### 主角反应
> "访问被拒绝？我被困在这里了？"
> （解锁后）"'欢迎您，旅人'...这个代号...和我有关系？"

#### 用途
- **即时效果**: 确认主角被困状态，增加紧张感
- **剧情推进**: 神秘人解锁后触发最终选择

---

### 2.3 窗帘与窗户

| 属性 | 内容 |
|------|------|
| **线索ID** | covered_window |
| **发现方式** | 尝试查看窗外 |
| **状态** | 被厚重窗帘遮挡 |
| **类型** | 氛围线索 |

#### 描述
落地窗被厚重的深色窗帘完全遮挡，从缝隙中无法看到外面。窗帘似乎被固定在窗框上，无法拉开。

#### 主角反应
> "窗帘被封死了...他们不想让我看到外面。"
> "这里是哪？地下？高楼？我完全不知道..."

#### 用途
- **即时效果**: 增加被困的压抑感
- **暗示**: 主角处于被控制状态
