## 视频生成要点

### 第一人称视角要求

| 要素 | 具体要求 |
|------|----------|
| 起始位置 | 走廊一端 |
| 运动轨迹 | 缓慢前进，可选择方向 |
| 运动速度 | 谨慎、缓慢 |
| 手部可见 | 可扶墙或自然下垂 |
| 画面稳定 | 稳定行走感 |

### 分镜要点

1. **进入镜头**（5-10秒）
   - 从门口进入走廊
   - 适应光线

2. **前进镜头**（15-20秒）
   - 沿走廊前进
   - 观察两侧门和标识

3. **选择镜头**（10-15秒）
   - 到达分岔路口
   - 环顾不同方向

---

## 注意事项

1. **氛围统一**：与其他场景的工业/现代风格保持一致
2. **灯光效果**：闪烁的灯光增加紧张感
3. **可选择性**：分岔路口需要有视觉区分
4. **透视感**：走廊需要有明显的透视延伸感
