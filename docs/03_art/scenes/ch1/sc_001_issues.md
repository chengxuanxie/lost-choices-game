## 问题5: 躺卧视角逻辑错误

| 问题ID | P005 |
|--------|------|
| **严重程度** | 🔴 高 - 影响所有躺着视角的关键帧 |
| **影响帧号** | Frame 001, 002, 003 |
| **问题描述** | 躺在床上的视角方向搞反了 |

**正确的躺卧视角逻辑**:
```
【人体工学分析】

人物躺在床上时：
- 头枕在床头（靠近床头柜的一端）
- 脚朝向床尾（靠近门的方向）
- 视线自然看向脚的方向（床尾方向）

因此躺卧视角应该看到：
- 正上方：天花板
- 脚的方向（床尾）：门、电子锁、部分窗户
- 需要转头才能看到：床头柜、台灯（在头的侧边）

正确的房间布局（俯视图）：
┌─────────────────────────────────────────┐
│ [墙 - 左侧]                               │
│  [神秘符号] ← 左墙高处                     │
│                                          │
│    ┌─────────┐         ┌─────────┐       │
│    │  窗户   │         │ 电子门  │       │
│    │(窗帘遮挡)│         │[电子锁] │       │
│    └─────────┘         └─────────┘       │
│                                          │
│  [简易桌椅]                               │
│                                          │
│        ┌───────────┐                     │
│  床头→ │    床     │ ←床尾               │
│        │  [主角]   │   (脚的方向)         │
│        └───────────┘                     │
│            ↑                             │
│     [床头柜+台灯]                         │
│                                          │
│                              [扬声器]     │
└─────────────────────────────────────────┘

主角躺着的视野（正确POV）：
┌─────────────────────────────────────────┐
│          [天花板]                         │
│                                          │
│  ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ │
│         ↑ 视线方向（看向床尾）             │
│                                          │
│     [门]              [窗户]              │
│   [电子锁]           (窗帘遮挡)           │
│                                          │
└─────────────────────────────────────────┘

注意：床头柜和台灯在头的侧后方，不在直接视野内
需要转头才能看到床头柜和台灯
```
