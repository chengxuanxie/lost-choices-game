# SC-002 可收集物品与线索

> 废弃仓库场景中的关键物品和线索设计

---

## 一、可收集物品

> **说明**：SC-002以对话为主，可收集物品较少，主要是环境互动和信息获取

### 1.1 仓库地图碎片（可选发现）

| 属性 | 内容 |
|------|------|
| **物品ID** | warehouse_map_fragment |
| **获取方式** | 在货架区域探索时发现 |
| **获取时间** | 约1分钟（探索阶段） |
| **外观** | 一张破旧的纸张，上面有部分地图标记 |

#### 内容
```
一张撕碎的纸张，上面是仓库的部分平面图。
可以看到一些标记：
- "A区" - 入口区域
- "B区" - 货架区域
- "C区" - 设备区
- 红色标记 - "紧急出口"
```

#### 主角反应
> "这是...这个仓库的地图？"
> "红色标记...是紧急出口的位置？"

#### 用途
- **即时效果**: 帮助了解仓库布局
- **可选**: 不影响主线剧情，但增加探索感

---

### 1.2 旧报纸碎片（可选发现）

| 属性 | 内容 |
|------|------|
| **物品ID** | old_newspaper_fragment |
| **获取方式** | 在货架上的箱子中发现 |
| **获取时间** | 约1分30秒（探索阶段） |
| **外观** | 一张泛黄的报纸碎片 |

#### 内容
```
报纸标题模糊可见：
"神秘组织活动频繁，警方展开调查"
日期：约3个月前

部分文字：
"...据知情人士透露，该组织代号'黄昏'..."
"...多名失踪案件可能与该组织有关..."
```

#### 主角反应
> "'黄昏'组织...那个神秘人提到过这个名字..."
> "这个组织...和我有什么关系？"

#### 用途
- **即时效果**: 增加"黄昏"组织的信息
- **剧情呼应**: 与SC-001神秘人的话呼应
- **伏笔**: 为后续剧情做铺垫

## 二、环境线索

### 2.1 仓库使用痕迹

| 属性 | 内容 |
|------|------|
| **线索ID** | warehouse_activity |
| **发现方式** | 观察环境 |
| **可见性** | 明显 |
| **类型** | 氛围线索 |

#### 描述
仓库虽然废弃，但某些区域有近期活动的痕迹：
- 地面上有较新的脚印
- 某些箱子被移动过
- 角落有临时堆放的物资

#### 主角反应
> "这里...最近有人活动过？"
> "是那个神秘人说的'接应者'吗？"

#### 用途
- **即时效果**: 增加紧张感
- **暗示**: 林晓薇已经在这里等待

---

### 2.2 紧急出口标识

| 属性 | 内容 |
|------|------|
| **线索ID** | emergency_exit_sign |
| **发现方式** | 观察仓库另一侧 |
| **可见性** | 在月光下可见 |
| **类型** | 导航线索 |

#### 描述
仓库另一侧的门上方有一个暗淡的紧急出口标识，门缝中透出城市夜晚的灯光。

#### 主角反应
> "那边是出口..."
> "门外的灯光...是城市的灯光吗？"

#### 用途
- **即时效果**: 指示离开方向
- **视觉引导**: 帮助玩家理解场景布局

---

### 2.3 林晓薇的装备

| 属性 | 内容 |
|------|------|
| **线索ID** | lin_xiaowei_gear |
| **发现方式** | 观察林晓薇 |
| **可见性** | 对话时可见 |
| **类型** | 角色线索 |

#### 描述
林晓薇身上有一些细节暗示她的身份：
- 腰间挂着一个小型手电筒
- 口袋里露出笔记本的一角
- 手腕上戴着实用的手表

#### 主角反应
> "她看起来...很专业。"
> "调查记者？她说的应该是真的..."

#### 用途
- **即时效果**: 增加林晓薇可信度
- **角色塑造**: 暗示她的职业背景

## 三、对话信息线索

### 3.1 林晓薇透露的信息

| 信息类型 | 内容 | 获取条件 |
|----------|------|----------|
| 基础信息 | 她是林晓薇，被神秘人派来接应 | 所有版本 |
| 身份信息 | 她是调查记者，正在调查"黄昏"组织 | 版本B/C |
| 主角信息 | 她认识"旅人"（过去的玩家） | 版本C |
| 追兵信息 | "黄昏"组织的人在追杀主角 | 所有版本 |

> **前置条件**：SC-001选择"在帮你之前，我想知道我是谁"

**林晓薇额外透露**：
