## 视觉描述

### 图标样式
- 线性图标风格
- 软盘造型（经典存档图标）
- 线条粗细：2px

### 结构描述
- 整体：圆角矩形
- 顶部：小矩形标签区域
- 底部：磁盘滑块区域
- 整体比例：约 4:3

### 颜色
- 默认颜色：#FFFFFF (白色)
