## 视觉描述

### 整体外观
- 圆角矩形按钮，圆角半径 12px
- 整体风格简约现代，与深色主题协调
- 无边框或极淡边框设计

### 背景填充
- 背景颜色：渐变色
  - 起始色：#D4A574 (温暖金)
  - 结束色：#E8C99B (浅金色)
  - 渐变方向：从左下到右上（135度）
- 可选：轻微的噪点纹理增加质感

### 边框
- 无明显边框
- 可选：1px 半透明边框 rgba(255,255,255,0.1)

### 阴影
- 无阴影（正常状态）

### 文字占位
- 文字颜色：#0A0A0A (深色)
- 文字居中显示
- 字体：Noto Sans SC Medium
- 字号：20px

## 技术要求

### 导出规格
- 格式：PNG-24 (支持透明)
- 色彩模式：sRGB
- 位深度：24位 + Alpha通道
- 导出倍率：@1x, @2x, @3x

### 九宫格切片
- 此按钮可考虑使用九宫格拉伸
- 切片区域建议：
  - 左边距：20px
  - 右边距：20px
  - 上边距：12px
  - 下边距：12px

### 文件命名
```
btn_lostchoices_primary_normal.png        # @1x
btn_lostchoices_primary_normal@2x.png     # @2x
btn_lostchoices_primary_normal@3x.png     # @3x
```
