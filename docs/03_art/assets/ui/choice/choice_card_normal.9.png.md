## 视觉描述

### 整体外观
- 圆角矩形卡片
- 圆角半径：12px
- 低调不抢眼，让文字成为焦点

### 背景
- 背景颜色：半透明深色
  - rgba(255,255,255,0.05)
- 或纯色：#252525

### 边框
- 1px 实线边框
- 边框颜色：#333333

### 内边距
- 上下：16px
- 左右：24px

## 技术要求

### 九宫格切片
- 使用 .9.png 格式
- 切片区域：
  - 左边距：24px（保留圆角和内边距）
  - 右边距：24px
  - 上边距：16px
  - 下边距：16px

### 切片示意
```
   24px      可拉伸      24px
    ↓         ↓           ↓
┌───────┬─────────────────┬───────┐

│       │                 │       │
```
