## 视觉描述

### 整体氛围
- 深色神秘氛围
- 与游戏"迷途抉择"主题呼应
- 不要过于抢眼，让UI和标题成为焦点

### 背景元素
- 基础色：深黑色 #0A0A0A
- 渐变：从中心向外微微变亮
- 纹理：轻微的噪点或颗粒感
- 光效：微弱的环境光或雾气效果

### 可选元素
- 模糊的城市轮廓（远景）
- 朦胧的光斑或光晕
- 抽象的线条或几何图形

### 氛围关键词
```
神秘 · 悬疑 · 电影感 · 深邃 · 期待
```

## 技术要求

### 导出规格
- 格式：JPG (质量85%) 或 WebP
- 色彩模式：sRGB
- 尺寸：1920×1080（可适当放大以支持高分屏）

### 性能考虑
- 文件大小控制在 200KB 以内
- 可考虑使用渐变+代码实现背景，减少图片体积

## 使用场景

- 游戏主菜单背景
- 可复用于暂停菜单背景（添加模糊效果）
