## 视觉描述

### 图标样式
- 线性图标风格
- 齿轮造型
- 圆形中心 + 外围齿
- 线条粗细：2px

### 结构描述
- 中心圆环：直径 8px，内圆直径 4px
- 外围齿：6-8个均匀分布
- 齿的形状：方形或圆角方形

### 颜色
- 默认颜色：#FFFFFF (白色)
