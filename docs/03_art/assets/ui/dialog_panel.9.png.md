## 视觉描述

### 整体外观
- 居中显示的模态对话框
- 圆角矩形面板
- 深色半透明背景
- 顶部可带标题栏

### 背景
- 背景颜色：深色
  - rgba(20,20,20,0.95)
- 可选：顶部标题栏使用稍亮的颜色

### 边框
- 1px 微弱边框
- 边框颜色：rgba(255,255,255,0.1)

### 圆角
- 圆角半径：12px

### 阴影
- 外阴影：0 4px 24px rgba(0,0,0,0.5)
- 模糊背景效果（毛玻璃）

## 技术要求

### 九宫格切片
- 使用 .9.png 格式
- 切片区域：
  - 左边距：16px
  - 右边距：16px
  - 上边距：16px（含标题栏区域）
  - 下边距：16px

### 尺寸范围
- 最小宽度：320px
- 最大宽度：600px
- 高度：根据内容自适应
