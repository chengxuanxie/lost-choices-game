## 视觉描述

### 图标样式
- 线性图标风格
- 锁头造型
- 线条粗细：2px

### 结构描述
- 锁身：圆角矩形
- 锁环：顶部弧形（闭合状态）
- 锁孔：中心小圆点或钥匙孔形状

### 颜色
- 默认颜色：#666666 (中灰)
- 可在代码中动态调整颜色
