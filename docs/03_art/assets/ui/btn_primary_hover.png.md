## 视觉描述

### 与正常状态差异
- 背景亮度提升 10-15%
- 添加发光效果：box-shadow: 0 0 20px rgba(212,165,116,0.4)
- 轻微放大效果（scale: 1.02，由程序实现）

### 背景填充
- 背景颜色：渐变色（亮度提升版）
  - 起始色：#DEB384
  - 结束色：#EED4AB

### 发光效果
- 外发光颜色：rgba(212,165,116,0.4)
- 发光半径：20px
- 发光位置：居中
