## 视觉描述

### 图标样式
- 线性图标风格
- 扬声器造型
- 线条粗细：2px

### 结构描述
- 扬声器主体：梯形
- 声波线：1-3条弧线

### 颜色
- 默认颜色：#FFFFFF (白色)

## 变体说明

| 变体名称 | 说明 |
|----------|------|
| icon_volume_high.svg | 高音量（3条声波） |
| icon_volume_medium.svg | 中音量（2条声波） |
| icon_volume_low.svg | 低音量（1条声波） |
| icon_volume_mute.svg | 静音（无声波 + X 或 斜线） |
