## 视觉描述

### 图标样式
- 线性图标风格
- 三条水平线（汉堡菜单）
- 线条粗细：2px
- 线条长度：18px
- 线条间距：6px
- 圆角：1px

### 颜色
- 默认颜色：#FFFFFF (白色)
- 可通过 CSS/程序改变颜色

## 技术要求

### 导出规格
- 格式：SVG（矢量格式）
