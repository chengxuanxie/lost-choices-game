## 视觉描述

### 图标样式
- 线性图标风格
- 心形造型
- 线条粗细：2px

### 结构描述
- 标准心形轮廓
- 稍微圆润的外观
- 可填充或镂空两种风格

### 颜色
- 默认颜色：#D4A574 (温暖金/品牌主色)
- 正向变化时：#D4A574 (金色)
- 负向变化时：#995B5B (暗红)

## 使用场景

- 选项卡片上显示好感度变化
  - [好感度+] 显示金色心形 + 数字
  - [好感度-] 显示暗红心形 + 数字
- 角色关系界面显示好感度等级
