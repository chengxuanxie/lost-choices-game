# 对话/字幕脚本目录

> 第一章视频对话和字幕脚本

---

## 目录结构

```
scripts/
└── ch1/                 # 第一章
    ├── ch1_sc_001.md     # SC-001 对话脚本
    ├── ch1_sc_002a.md    # SC-002 对话脚本 (分支A)
    ├── ...               # 其他场景
    └── ch1_ending_*.md   # 结局对话脚本
```

---

## 文档格式

每个对话脚本包含：

1. **YAML元数据** - 视频信息配置
2. **字幕/配音清单** - 概览统计
3. **详细脚本** - 按时间段的对话和独白
4. **玩家选择** - 选项内容和效果
5. **SRT导出** - 字幕文件格式
6. **AI配音配置** - TTS参数
