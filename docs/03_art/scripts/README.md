## 文档格式

每个对话脚本包含：

1. **YAML元数据** - 视频信息配置
2. **角色配置** - 声音类型和特点
3. **详细脚本** - 按时间段的对话和独白
4. **玩家选择** - 选项内容和效果
5. **选择触发对话** - 选择后的对话
6. **音效时序** - 环境音效和效果音
7. **字幕文件导出** - SRT格式
8. **配音注意事项** - 情绪、语速、特殊处理

## SRT字幕格式

```
1
00:00:05,000 --> 00:00:15,000
字幕内容第一行

2
00:00:20,000 --> 00:00:30,000
字幕内容第二行
```

---

## 配音工作流程

1. 根据脚本选择合适的配音角色
2. 使用Edge-TTS或其他TTS工具生成配音
3. 按照配音注意事项调整情绪和语速
4. 添加内心独白的回响效果
5. 按音效时序添加环境音效
6. 导出SRT字幕文件
