## 📌 整体风格定位

### 核心风格关键词
- **沉浸式** - 第一人称视角，强调代入感
- **电影感** - 专业拍摄手法，电影级画面
- **写实** - 真人表演，真实场景
- **情绪化** - 通过光影和色彩传递情感

### 视觉定位
打造电影级的互动叙事体验，通过高质量的视频拍摄和专业的后期制作，为玩家呈现沉浸式的第一人称视角故事。

## 🎬 视频拍摄规范

### 拍摄风格

#### 视角要求
| 要素 | 规范 |
|------|------|
| 视角类型 | 第一人称视角（POV） |
| 镜头高度 | 眼睛水平高度 |
| 镜头运动 | 模拟头部运动，自然流畅 |
| 稳定度 | 轻微自然晃动，避免过度抖动 |

#### 拍摄设备
| 设备 | 要求 | 说明 |
|------|------|------|
| 摄影机 | 4K及以上 | 推荐：索尼A7系列 / Blackmagic |
| 镜头 | 广角镜头 | 24-35mm，模拟人眼视野 |
| 稳定器 | 必需 | POV专用稳定设备 |
| 收音设备 | 专业麦克风 | 指向性麦克风 + 无线领夹麦 |

### 画面规范

#### 分辨率与比例
| 参数 | 规范 |
|------|------|
| 分辨率 | 3840×2160 (4K) / 1920×1080 (1080p) |
| 宽高比 | 16:9 |
| 帧率 | 24fps（电影感） / 30fps（流畅感） |
| 色深 | 10bit |
| 色彩空间 | Rec.709 / Log拍摄 |

## 🖥️ UI设计规范

### 整体UI风格
- **设计语言**: 简约现代、沉浸式
- **视觉层次**: 低饱和度背景 + 高对比度内容
- **动效风格**: 流畅、自然、不干扰视频

### UI配色

#### 深色主题（默认）
```css
/* 主背景 */
--bg-primary: #0A0A0A;
--bg-secondary: #1A1A1A;
--bg-tertiary: #2A2A2A;

/* 文字颜色 */
--text-primary: #FFFFFF;
--text-secondary: #B0B0B0;
--text-disabled: #666666;

/* 强调色 */
--accent-primary: #D4A574;
--accent-secondary: #5B7C99;
--accent-warning: #E85A4F;
--accent-success: #4CAF50;

/* 边框与分割线 */
--border-light: #333333;
--border-dark: #222222;
```

#### 浅色主题（可选）
```css
/* 主背景 */
--bg-primary: #F5F5F5;

--bg-tertiary: #E8E8E8;

/* 文字颜色 */

/* 强调色保持一致 */
```

## 🎭 场景美术

### 场景类型与风格

#### 场景分类
| 类型 | 视觉风格 | 光线特点 | 色调倾向 |
|------|----------|----------|----------|
| 居家场景 | 温馨舒适 | 柔和自然光 | 暖色调 |
| 办公场景 | 现代专业 | 冷白光源 | 中性偏冷 |
| 室外日景 | 明亮开阔 | 自然阳光 | 清新明亮 |
| 室外夜景 | 神秘氛围 | 人工光源 | 冷色调 |
| 特殊场景 | 剧情相关 | 氛围光 | 情绪导向 |

### 场景设计原则

1. **第一人称适配**: 所有场景设计需考虑POV视角
2. **视觉焦点**: 引导玩家视线关注关键元素
3. **氛围一致性**: 场景风格与剧情情绪匹配
4. **细节层次**: 重要区域高细节，背景适度简化

## ✨ 后期制作规范

### 调色风格

#### 基础调色参数
```
整体风格: 电影感调色

对比度: +10 ~ +15
饱和度: -5 ~ -10 (轻微降低，营造质感)
色温: 根据场景调整
高光: -10 ~ -20 (保留高光细节)
阴影: +5 ~ +10 (提升暗部细节)
```

#### 场景调色预设
| 场景类型 | 色温 | 色调 | 对比度 |
|----------|------|------|--------|
| 温馨室内 | +15 | +5 | +10 |
| 紧张场景 | -10 | +10 | +20 |
| 回忆场景 | +5 | -10 | -5 |
| 危机场景 | -20 | +15 | +25 |

### 转场效果

#### 视频转场类型
| 转场类型 | 时长 | 使用场景 |
|----------|------|----------|
| 淡入淡出 | 0.5-1秒 | 场景切换 |
| 硬切 | 0秒 | 快节奏叙事 |
| 闪白 | 0.3秒 | 回忆/想象 |
