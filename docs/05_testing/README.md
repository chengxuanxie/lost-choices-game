### Bug优先级定义

| 优先级 | 定义 | 响应时间 |
|--------|------|----------|
| P0 - 致命 | 游戏无法启动/无法进行 | 立即修复 |
| P1 - 严重 | 主要功能受阻 | 24小时内 |
| P2 - 一般 | 非主要功能问题 | 3天内 |
| P3 - 轻微 | 小问题/优化建议 | 下个版本 |
