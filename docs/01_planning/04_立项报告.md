## 📊 项目摘要

### 项目定位
《迷途抉择》是一款第一人称视角的视频互动游戏，玩家以主角视角沉浸式体验故事，通过选择对话选项影响剧情走向。游戏采用真人实拍视频，首发Web平台实现即点即玩。

### 核心卖点
1. **第一人称沉浸视角** - 区别于市场主流产品，提供更强的代入感
2. **Web即点即玩** - 降低用户门槛，便于传播和分享
3. **本土化内容** - 贴近国内用户文化偏好和审美
4. **合理定价** - 首章免费+按章节付费，性价比优势

### 市场机会
根据市场调研，互动游戏市场规模持续增长（年增长率约24%），国内优质互动游戏稀缺，存在明显市场空白。参考同类产品《隐形守护者》首月销量超50万份，证明国内市场潜力巨大。
