## 🔧 技术方案

### 混合视觉方案（已采用）
```
关键帧图片 + 着色器效果  ←  主力（成本节省87%）
        ↓
短视频  ←  仅关键剧情节点（~12个）

详情: docs/03_art/video_production/hybrid_visual_solution.md
```

### 一场景一参考图规范
```
每个场景 = 1张核心参考图
所有关键帧/视频 → 基于这张参考图生成

详情: docs/03_art/03_AI视频制作指南.md
```

## 📞 常见问题

### Q: 如何开始生成关键帧？
1. 打开 `docs/03_art/scenes/ch1/sc_001_神秘房间.md`
2. 查看"三、关键帧需求"部分
3. 使用 img_gen skill，复制Prompt模板

### Q: 如何运行测试？
```
Godot → 打开 scenes/test_runner.tscn → F5运行
```

### Q: 如何添加新场景？
1. 参考 `docs/03_art/scenes/ch1/sc_001_神秘房间.md` 格式
2. 生成1张核心参考图
3. 编写镜头分解和关键帧需求
