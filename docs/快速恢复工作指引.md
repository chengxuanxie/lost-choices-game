## 🛠 Claude Skills 使用

| Skill | 用途 | 触发示例 |
|-------|------|----------|
| `img_gen` | 即梦4.0图像生成 | "生成关键帧图片..." |
| `img_upload` | 火山引擎TOS上传 | "上传图片到TOS" |
| `video_gen` | 即梦视频生成 | ⚠️ 费用高，每次需确认 |
| `motion_comic` | 动态漫视频生成 | ✅ 已用于SC-001 |

### 关键帧生成规范

> **链式参考策略**：连续视角/动作的关键帧，应以前一帧生成结果作为参考图

| 策略 | 图标 | 适用场景 |
|------|------|----------|
| 初始参考图 | 📍 | 场景起点、独立镜头 |
| 链式参考 | 🔄 | 连续动作、视角渐变 |
| 无参考图 | ⚡ | 特殊效果、抽象画面 |

**详细规范**: `docs/03_art/03_AI视频制作指南.md` 第三节
