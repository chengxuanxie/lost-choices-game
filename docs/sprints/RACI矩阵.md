# 📋 RACI矩阵

> 《迷途抉择》项目角色职责分工表

---

## 说明

- **R (Responsible)**: 执行者，负责完成任务
- **A (Accountable)**: 问责者，对任务结果负责，有决策权
- **C (Consulted)**: 咨询者，提供意见和建议
- **I (Informed)**: 知情者，被告知结果
