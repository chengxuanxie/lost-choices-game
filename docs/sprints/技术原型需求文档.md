## 一、原型目标

### 1.1 验证目标

技术原型需要在M2里程碑（2026-03-15）前验证以下关键技术可行性：

| 序号 | 验证项 | 重要性 | 验证标准 |
|------|--------|--------|----------|
| 1 | Godot Web导出 | 🔴 关键 | 可成功导出Web版本并运行 |
| 2 | 视频播放性能 | 🔴 关键 | 视频起播<2秒，播放流畅 |
| 3 | 浏览器兼容性 | 🔴 关键 | Chrome/Firefox/Safari/Edge均可用 |
| 4 | 剧情分支系统 | 🟡 重要 | 可实现简单的分支选择 |
| 5 | 存档系统 | 🟡 重要 | IndexedDB存档读写正常 |

### 1.2 原型范围

**包含**：
- 基础视频播放功能
- 简单的UI界面
- 2-3个剧情节点的分支演示
- 存档/读档功能演示
- 基础音效播放

**不包含**：
- 完整游戏内容
- 真人视频素材（使用占位视频）
- 完整UI设计
- 后端服务
- 支付系统

## 二、功能需求

### 2.1 核心功能清单

| 功能ID | 功能名称 | 优先级 | 说明 |
|--------|----------|--------|------|
| F001 | 视频播放 | P0 | 播放MP4格式视频 |
| F002 | 视频切换 | P0 | 视频间平滑切换 |
| F003 | 选择系统 | P0 | 显示选项并响应点击 |
| F004 | 分支跳转 | P0 | 根据选择跳转不同视频 |
| F005 | 存档功能 | P1 | 保存当前进度 |
| F006 | 读档功能 | P1 | 恢复保存的进度 |
| F007 | 音效播放 | P2 | 播放背景音乐和音效 |
| F008 | 字幕显示 | P2 | 显示视频字幕 |

### 2.2 功能详细说明

#### F001 视频播放

**需求描述**：
- 支持MP4格式视频播放
- 支持播放、暂停、继续操作
- 显示播放进度
- 支持音量调节

**验收标准**：
- [ ] 视频可正常播放
- [ ] 播放器控制按钮可用
- [ ] 进度条可拖动
- [ ] 音量可调节

#### F002 视频切换

**需求描述**：
- 当前视频播放结束后自动切换到下一视频
- 选择选项后切换到对应视频
- 切换时有过渡动画（淡入淡出）

**验收标准**：
- [ ] 视频切换延迟<500ms
- [ ] 切换动画流畅
- [ ] 无明显卡顿或闪烁

#### F003 选择系统

**需求描述**：
- 视频播放到选择点时显示选项
- 支持2-4个选项
- 鼠标/触屏可点击
- 有悬停和点击反馈

**验收标准**：
- [ ] 选项正确显示
- [ ] 点击响应正常
- [ ] 有视觉反馈

#### F004 分支跳转

**需求描述**：
- 根据选择的选项跳转到对应的剧情节点
- 记录玩家的选择历史
- 支持"返回上一节点"功能（调试用）

**验收标准**：
- [ ] 分支跳转逻辑正确
- [ ] 选择历史可追溯
- [ ] 返回功能可用

#### F005 存档功能

**需求描述**：
- 点击存档按钮保存当前进度
- 显示存档成功提示
- 使用IndexedDB存储

**验收标准**：
- [ ] 存档成功并显示提示
- [ ] 数据正确存入IndexedDB
- [ ] 可存储多个存档位

#### F006 读档功能

**需求描述**：
- 显示存档列表
- 选择存档后恢复进度
- 显示存档信息（时间、进度）

**验收标准**：
- [ ] 存档列表正确显示
- [ ] 读档后进度恢复正确
- [ ] 读档失败有错误提示

## 五、性能需求

### 5.1 性能指标

| 指标 | 目标值 | 测试方法 |
|------|--------|----------|
| 首屏加载时间 | < 5秒 | Chrome DevTools |
| 视频起播时间 | < 2秒 | 计时测试 |
| 视频切换延迟 | < 500ms | 计时测试 |
| 内存占用 | < 300MB | 任务管理器 |
| 帧率 | ≥ 30fps | Godot调试器 |

### 5.2 兼容性要求

| 浏览器 | 版本要求 | 测试重点 |
|--------|----------|----------|
| Chrome | 90+ | 主要测试对象 |
| Firefox | 88+ | 视频播放兼容性 |
| Safari | 14+ | 视频格式兼容性 |
| Edge | 90+ | 基础功能 |
