## 五、快速开始

### 运行游戏
```bash
# 在Godot中打开项目
# 运行 scenes/test/hybrid_game.tscn
```

### 加载剧情
游戏会自动加载 `chapter_01_hybrid.json` 作为剧情数据。

### 测试特定场景
在Godot中修改 `hybrid_game.gd` 的 `start_node` 为目标节点ID。

## 六、常见问题

### Q: 视频无法播放？
A: 检查 `chapter_01_hybrid.json` 中的路径是否正确，使用 `res://` 前缀

### Q: 关键帧不显示？
A: 确保图片放在 `res://assets/keyframes/ch1/` 目录下

### Q: 如何测试结局？
A: 通过游戏内选择特定选项触发，或直接跳转到对应结局节点
