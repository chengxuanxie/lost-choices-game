## 📊 项目概览

| 项目信息 | 内容 |
|---------|------|
| **项目名称** | 《迷途抉择》(Lost Choices) |
| **项目类型** | 第一人称视频互动游戏 |
| **开发引擎** | Godot 4.3+ |
| **目标平台** | Web网页端（首发），后续Steam/移动端 |
| **运行模式** | 单机模式（纯前端，无需服务器） |
| **团队规模** | 核心团队11人 + 外部资源 |
| **当前状态** | 🟡 M3第一章制作阶段进行中 |
| **开始日期** | 2026-02-20 |
| **预计发布** | 2026-08-31 |
| **项目预算** | 354万元人民币（单机模式节省47.5万） |

### 关键决策记录

| 决策ID | 决策内容 | 决策日期 | 决策依据 | 影响 |
|--------|----------|----------|----------|------|
| D001 | 选择Godot 4.x作为游戏引擎 | 2026-02-20 | 开源免费、Web支持好 | 节省授权费约2万/年 |
| D002 | 首发Web平台 | 2026-02-20 | 降低用户门槛、便于传播 | 需重点优化视频加载 |
| D003 | 采用第一人称视角 | 2026-02-20 | 差异化竞争、沉浸感强 | 需特殊拍摄设备和技术 |
| D004 | 首章免费+章节付费模式 | 2026-02-20 | 降低决策门槛、持续收入 | 影响定价策略 |
| D005 | 采用单机模式 | 2026-02-28 | 降低开发门槛和成本 | 节省云服务成本约47.5万 |
| D006 | 采用居家办公模式 | 2026-02-26 | 节省成本、灵活高效 | 节省办公场地租金约12万 |

## 💡 使用说明

### 新建文档
1. 进入对应阶段的文件夹
2. 复制 `templates/` 下的模板
3. 根据实际需求填写内容
4. 更新本文档的文档导航和更新日志

### 查看进度
- 各阶段文件夹内包含 `README.md` 说明当前阶段完成情况
- 使用 `checklists/` 下的清单核对工作完整性
- 查看 `sprints/` 了解当前迭代进展

### 文档协作
- 使用版本控制系统（Git）管理文档版本
- 重大修改请在更新日志中记录
- 定期进行文档评审会议

### 项目交接说明

**如需更换团队或人员变动，请按以下步骤交接：**

1. **阅读核心文档**（按顺序）：
   - 本文档（plan.md）- 了解项目整体状态
   - 立项报告 - 了解项目背景和决策依据
   - 项目计划书 - 了解详细计划和预算
   - 技术方案文档 - 了解技术架构
   - GDD - 了解游戏设计

2. **确认关键信息**：
   - 当前里程碑和任务状态
   - 风险监控状态
   - 预算执行情况
   - 团队配置状态

3. **联系关键人员**：
   - 参考上方联系人列表
   - 安排交接会议

4. **检查代码仓库**：
   - Git仓库状态
   - 分支策略
   - 最新提交记录
