## 📊 项目概览

| 项目信息 | 内容 |
|---------|------|
| **项目名称** | 《迷途抉择》(Lost Choices) |
| **项目类型** | 第一人称视频互动游戏 |
| **开发引擎** | Godot 4.3+ |
| **目标平台** | Web网页端（首发），后续Steam/移动端 |
| **运行模式** | 单机模式（纯前端，无需服务器） |
| **团队规模** | 核心团队11人 + 外部资源 |
| **当前状态** | 🟡 M1立项阶段进行中 |
| **开始日期** | 2026-02-20 |
| **预计发布** | 2026-08-31 |
| **项目预算** | 354万元人民币（单机模式节省47.5万） |

### 关键决策记录

| 决策ID | 决策内容 | 决策日期 | 决策依据 | 影响 |
|--------|----------|----------|----------|------|
| D001 | 选择Godot 4.x作为游戏引擎 | 2026-02-20 | 开源免费、Web支持好 | 节省授权费约2万/年 |
| D002 | 首发Web平台 | 2026-02-20 | 降低用户门槛、便于传播 | 需重点优化视频加载 |
| D003 | 采用第一人称视角 | 2026-02-20 | 差异化竞争、沉浸感强 | 需特殊拍摄设备和技术 |
| D004 | 首章免费+章节付费模式 | 2026-02-20 | 降低决策门槛、持续收入 | 影响定价策略 |
| D005 | 采用单机模式 | 2026-02-28 | 降低开发门槛和成本 | 节省云服务成本约47.5万 |
| D006 | 采用居家办公模式 | 2026-02-26 | 节省成本、灵活高效 | 节省办公场地租金约12万 |
